# AI Context

This folder contains context intended to help AI coding agents work safely and consistently on the project.

Use it for prompts, architectural summaries, coding conventions, domain assumptions, agent instructions, and any project knowledge that should be reused across AI-assisted development sessions.

## Architecture Decisions & Change Requests
To maintain a historical log of design decisions and requirement modifications:
- **Architecture Decision Records (ADRs)** are located in [docs/adr/](file:///c:/MyProjects/codex/Matrimitra/docs/adr).
- **Change Requests (CRs)** are located in [docs/cr/](file:///c:/MyProjects/codex/Matrimitra/docs/cr).

AI agents must consult the active ADRs and approved CRs before proposing or executing modifications to specifications or code.
